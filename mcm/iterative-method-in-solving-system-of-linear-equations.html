大一线代学的高斯消元法不适合高阶线性方程组。因此发展新的方法——迭代法。

# 迭代法的基本概念

设方程组 Ax=fAx=fAx=f 有唯一解 x∗x^*x∗。可将方程组变形为：

x=Bx+fx=Bx+fx=Bx+f

由此建立迭代公式：

x(k+1)=Bx(k)+f(k=0,1,2,...)x^{(k+1)}=Bx^{(k)}+f \quad (k=0,1,2,...)x(k+1)=Bx(k)+f(k=0,1,2,...)

再给定 x(0)x^{(0)}x(0)，即可得到近似解序列 {x(k)}\{x^{(k)}\}{x(k)}。

若 lim⁡k→∞x(k)=x∗\lim\limits_{k\to\infty}x^{(k)}=x^*k→∞lim​x(k)=x∗，有 x∗=Bx∗+fx^*=Bx^*+fx∗=Bx∗+f（x∗x^*x∗为不动点），则称迭代法是收敛的，否则是发散的。

BBB 称为迭代矩阵，BBB 的取法不同，就是各种迭代法的区别。各种方法是否收敛？速度如何？如何进行误差估计？这就是我们要研究的问题。

迭代法适用于解大型稀疏方程组。

# 雅可比迭代法

最简单的方法是将每个方程组左边留一个变量，其他全部丢到右边，这就是一个最简单的迭代矩阵。

具体的来说，对于方程组中第 iii 个方程，

∑j=1naijxj=bi(i=1,2,...,n)\sum_{j=1}^n a_{ij}x_j = b_i \quad (i=1,2,...,n)j=1∑n​aij​xj​=bi​(i=1,2,...,n)

我们把 aiixia_{ii} x_iaii​xi​ 留在左边，其他丢到右边，再两边同时除以 aiia_{ii}aii​，得到迭代公式为：

xi(k+1)=1aii[bi−∑j=1i−1aijxj(k)−∑j=i+1naijxj(k)]x_i^{(k+1)} = \frac{1}{a_{ii}} [b_i - \sum_{j = 1}^{i-1}a_{ij}x_j^{(k)} - \sum_{j = i+1}^{n}a_{ij}x_j^{(k)}]xi(k+1)​=aii​1​[bi​−j=1∑i−1​aij​xj(k)​−j=i+1∑n​aij​xj(k)​]

（迭代公式中后面两项其实可以合并为一个求和符号，但是这里分开写，是为了给后面做铺垫）

初始点任取，进行迭代即可。

不过呢，这玩意不一定收敛。

# 高斯——赛德尔迭代法

注意到，如果上面如果收敛，一般来说，新值比旧值优秀。
所以我们尽可能使用新的值，把第 k+1k+1k+1 次迭代前面算出来的数用到该次迭代中（迭代表达式中只改了一个上标）：

xi(k+1)=1aii[bi−∑j=1i−1aijxj(k+1)−∑j=i+1naijxj(k)]x_i^{(k+1)} = \frac{1}{a_{ii}} [b_i - \sum_{j = 1}^{i-1}a_{ij}x_j^{(k+1)} - \sum_{j = i+1}^{n}a_{ij}x_j^{(k)}]xi(k+1)​=aii​1​[bi​−j=1∑i−1​aij​xj(k+1)​−j=i+1∑n​aij​xj(k)​]

理论上，如果使用两个方法都收敛，高斯——赛德尔方法会略快于雅可比迭代。

# 迭代的收敛性

在基本概念中提到，迭代即是

lim⁡k→∞X(k)=X∗⇔lim⁡k→∞∥X(k)−X∗∥2=0 \lim_{k \to \infty}X^{(k)} = X^* \Leftrightarrow \lim_{k \to \infty}\| X^{(k)} - X^* \|_2 = 0 k→∞lim​X(k)=X∗⇔k→∞lim​∥X(k)−X∗∥2​=0

令 ε(k)=X(k)−X∗\varepsilon^{(k)} = X^{(k)} - X^*ε(k)=X(k)−X∗，则上述结论可以写为

lim⁡k→∞X(k)=X∗⇔lim⁡k→∞∥ε(k)∥2=0 \lim_{k \to \infty}X^{(k)} = X^* \Leftrightarrow \lim_{k \to \infty}\| \varepsilon^{(k)} \|_2 = 0 k→∞lim​X(k)=X∗⇔k→∞lim​∥ε(k)∥2​=0

（利用向量范数的等价性，我们还可以进一步推出上述结论对任意范数同样成立）

接下来，我么换一个角度，看从迭代法的定义还能推出什么关于 ε(k)\varepsilon^{(k)}ε(k) 的结论：

X(k+1)=BX(k)+f(k=0,1,2,...)X∗=BX∗+fX(k+1)−X∗=B(X(k)−X∗)\begin{aligned} X^{(k+1)} &= BX^{(k)}+f \quad (k=0,1,2,...) \\ X^* &= BX^* + f \\ X^{(k+1)} - X^* &= B(X^{(k)}-X^*) \end{aligned} X(k+1)X∗X(k+1)−X∗​=BX(k)+f(k=0,1,2,...)=BX∗+f=B(X(k)−X∗)​

则有：

ε(k)=Bε(k−1)(k=1,2,3,...) \varepsilon^{(k)} = B \varepsilon^{(k-1)} \quad (k=1,2,3,...) ε(k)=Bε(k−1)(k=1,2,3,...)

ε(k)=Bε(k−1)=B2ε(k−2)=⋯=Bkε(0)\varepsilon^{(k)} = B \varepsilon^{(k-1)} = B^2 \varepsilon^{(k-2)} = \cdots = B^k \varepsilon^{(0)} ε(k)=Bε(k−1)=B2ε(k−2)=⋯=Bkε(0)

# 矩阵的谱

定义 矩阵的谱 chB\rm{ch}\;BchB （或σ(B)\sigma(B)σ(B)）就是矩阵特征值的集合 {λ1,λ2,...,λn}\{\lambda_1, \lambda_2, ..., \lambda_n\}{λ1​,λ2​,...,λn​}。

定义 矩阵 BBB 的谱半径 ρ(B)=max⁡1≤k≤n∣λk∣\rho(B) = \max\limits_{1\leq k\leq n}|\lambda_k|ρ(B)=1≤k≤nmax​∣λk​∣

那么，关于在 范数的相容性 中提到的最大特征值的结论都可以套过来：

结论 1：当 BBB 是对称矩阵时，∥B∥2=ρ(B)\|B\|_2 = \rho(B)∥B∥2​=ρ(B)
结论 2：对于任意范数都有：ρ(B)≤∥B∥2\rho(B) \leq \|B\|_2ρ(B)≤∥B∥2​

定理：

lim⁡k→∞ε(k)=0⇔ρ(B)<1\lim_{k\to\infty}\varepsilon^{(k)} = \mathbf{0} \Leftrightarrow \rho(B) < 1 k→∞lim​ε(k)=0⇔ρ(B)<1
