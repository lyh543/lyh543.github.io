# 甚至可以爆 long long （指忘取模）

快速幂（或类似物）对 ans 取模，没对 temp 取模。

ans = ans * temp % mod;
temp *= mod;

还有加法的时候忘了优先级。

C[i][j] = C[i-1][j-1] + C[i-1][j] % mod
