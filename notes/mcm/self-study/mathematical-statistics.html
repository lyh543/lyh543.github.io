# 秩和检验

秩 Rank：排序后的名次

用于判断两个分布有无显著差异。

方法是，将两个分布的数混起来然后排序。排完序以后，将其中一个分布的名次数加起来，称作秩和。

然后根据 n1n_1n1​、n2n_2n2​、ttt 查表。如果秩和在中间部分，则可认为无明显差异。

# Bootstrap

这个是统计学的 Bootstrap，不是网页 css 框架的那个。

Bootstrap 是在样本点不够的情况下，从已知的信息中（有放回地）抽抽抽。抽出来的数每 nnn 个数分为一组，美其名曰，一组容量为 nnn 的样本。

对于每一组样本都可以算出一个随机变量，就可以对算出来的所有随机变量进行计算，得到该随机变量的均方误差、标准误差、区间估计等信息。

这种方法的好处在上面也就提到了，在样本点较少的情况下会有奇效。

这种方法需要在计算机上做大量的计算，随着计算机威力的增长，它也成为了流行的方法。

# 非参数 Bootstrap

非参数 Bootstrap 是指，分布函数未知，仅知道一堆样本点。现在要计算某估计量 θ\thetaθ （这里估计量甚至可以取中位数）的标准差 D(θ^)\sqrt{D(\hat\theta)}D(θ^)​、均方误差 MSE=E[(M−θ)2]MSE=E[(M-\theta)^2]MSE=E[(M−θ)2] 和置信区间等。

那么就只需要在给出的样本点里面抽抽抽就可以了。

我们在给定的样本中有放回地抽 BBB 个大小为 nnn 的样本（共抽 BnBnBn 次），对于每个样本，计算出 θ\thetaθ 的估计 θ^i∗\hat\theta^*_iθ^i∗​，则 θ\thetaθ 的标准误差的 Bootstrap 估计为

σ^θ^=1B−1∑i=1B(θ^i∗−θ‾∗)2\hat\sigma_{\hat\theta}=\sqrt{\frac{1}{B-1}\sum_{i=1}^B(\hat\theta^*_i-\overline\theta^*)^2}σ^θ^​=B−11​i=1∑B​(θ^i∗​−θ∗)2​

其中 θ‾∗\overline\theta^*θ∗ 是原样本的 θ\thetaθ 的估计值。

# 多元线性回归

多元线性回归分析的模型是：

{y=β0+β1x1+⋯+βmxm+εε∼N(0,σ2)\begin{cases} y = \beta_0 + \beta_1x_1 + \cdots + \beta_mx_m + \varepsilon \\ \varepsilon \sim N(0, \sigma^2) \end{cases}{y=β0​+β1​x1​+⋯+βm​xm​+εε∼N(0,σ2)​

其中 βi\beta_iβi​ 为未知系数，称为回归系数。

现在测得 nnn 个独立的 [y,x1,x2,⋯,xm][y, x_1, x_2, \cdots, x_m][y,x1​,x2​,⋯,xm​] 的观测值，第 iii 个为 [bi,ai1,ai2,⋯,aim][b_i, a_{i1}, a_{i2}, \cdots, a_{im}][bi​,ai1​,ai2​,⋯,aim​]。

mmm 个待估计参数、nnn 组观测值。

  1. 第一步，进行参数估计。即以最小二乘法的原理，按公式计算得 mmm 个 β\betaβ，以及残差（算得的 βm\beta_mβm​ 和每组的 xmx_mxm​ 计算出的 ym′y_m'ym′​ 减去 ymy_mym​ 的差，即一个误差）
  2. 第二步，验证回归模型的正确性（即，是不是线性回归模型）。判断标准是 βi\beta_iβi​ 是不是全为 0。一种思想是运用 FFF 检验。另一种思想就是计算副判定系数 R2R^2R2。
  3. 第三步，对回归系数进行假设检验、区间估计。这里需要判定的是，βi\beta_iβi​ 中哪些是 0。
  4. 最后对回归模型进行预测，并给出区间估计。
