连接点

编辑连接线线条、箭头或点 - Visio - Microsoft Support

连接点是形状上的特殊点，可以向它“粘附”连接线和其他形状。 当将连接线或形状粘附至连接点时，即使某个形状发生移动，该连接线和连接点都保持连接状态。

除了使用系统自带的连接点，我们还可以为自己的图形添加连接点。

更改网格大小

在Visio 中显示、隐藏或更改网格 - Visio - Microsoft Support
视频：更改网格间距大小 - Visio - Microsoft Support

上面的添加连接点没有参考线容易弄歪，我们可以使用网格作为参考线。

如果网格太宽或者太窄，我们还可以自己进行调整。
