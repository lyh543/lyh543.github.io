与他人共享自己的 Azure 订阅

2020-10-31
  • Microsoft

目的

本文用于将 A（文中用 lyh543@outlook.com 作示范）的 Azure 订阅共享给 B（文中用 lxl361429916@live.com 作示范），共享以后，B 可以使用 A 的 Azure 订阅的额度创建资源。

图图图

前提条件

  • A 拥有 Azure 的订阅且未过期
  • B 可以使用下面的邮箱之一：
    • Azure 生成的邮箱账户
    • 已经存在，但还没有注册微软账户或启用 Azure 的邮箱
    • 已注册微软账户并启用 Azure 的邮箱（本文将演示这种情况）

第一步：A 邀请 B 加入自己的目录

A 登录 Azure 并进入 Azure 门户，从侧边栏进入 Azure Active Directory - 用户。

新建用户，其中“创建用户”和“邀请用户”可以选后者，然后输入 B 的邮箱，其他默认/不填。

需要注意的是，这个添加的账户是隶属于 xxx.onmicrosoft.com 的工作邮箱，而不是个人的微软邮箱。

第二步：B 同意邀请

此时 B 应该收到了邀请邮件，点击 Accept invitation 进行登录。

登录的时候请注意，如果要选择“工作或学校账户”还是“个人账户”，请选择“工作或学校账户”，因为这是隶属于 xxx.onmicrosoft.com 的工作邮箱。
