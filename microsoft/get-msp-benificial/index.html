获取 MLSA (原 MSP) 的 Office 365 和 Visual Studio 订阅权益

2020-06-01
  • Microsoft

以下内容如有不准确，请以微软妹为准。

MLSA（原为 Microsoft Student Partner，MSP）的全称为 Microsoft Learn Student Ambassadors（Microsoft Learn 学生大使），旨在寻找对创造价值、建立充满活力的社区，以及与同学分享最新技术充满热情的学生领导者。

成为一名 MLSA 后，能够获得 Office 365 订阅和 Visual Studio 订阅，后者包括每月 150 美元（或等值的其他币种）的 Azure 额度、微软全系产品的数个激活码、24 个含有 Office 365 订阅的子账户。晋升为 beta 级别后，还能获得 T 恤和大量的小纪念品以供举办活动。

获取 Office 365

成为 MSP 后，会获得一个 @studentpartner.com 邮箱（还不清楚 MSP 改名为 MLSA 后，邮箱域名是否会变化），在 MLSA 退役后邮箱会被注销。

用该邮箱登录 https://office.com/，就可以获取 Office 365 了。

用该邮箱登录 https://azure.microsoft.com/ ，就能获得每年 120 刀的 Azure 额度，这是学生账户（非中国区）的权益。注意选择的时候最好不要选择中国，以免后面麻烦。

升级为 alpha 2 和 beta

MLSA 的评级影响着权益：

  • alpha 能获得 Office 365 订阅和每年 120 刀的 Azure 额度，直接使用 @studentambassadors.com 邮箱登录 https://azure.com/ 和 https://office.com/ 即可获取
  • alpha 2 能获得 Visual Studio 订阅，包括每月 150 美元的 Azure 额度等
  • beta 能获得 T 恤和大量的小纪念品
  • gold 的忘了，貌似有奖杯

晋升的大致规则如下：

  • alpha 在 Microsoft Learn 完成一个 Learning Path / 学习路径 即可晋升至 alpha 2
    • 注意是一个学习路径，而不是一个模块
    • 完成学习路径会获得奖杯（在个人配置文件的成就处可见）
    • 需在 Teams 中给出的 Microsoft Form 填写记录，一般一个晚上审核通过
  • alpha 2 举办一次至少 15 人参加的 live-event 技术分享会即可晋升至 beta
  • beta 晋升至 gold，需成为 MSP 核心成员

一般来说，每位新 MLSA 需尽快升至 alpha 2（一般为一周以内）；在一年内升至 beta。

获取 Visual Studio 订阅

Visual Studio 订阅在升级到 beta 以后即可获得，在 https://my.visualstudio.com/ 查看。

如果登录的时候出现了下图登录失败，可能是账户进行了异常操作（如申请秘钥、修改备用账户等），24 小时内会自动解封。

获取 Microsoft 365 开发人员订阅 (E5)

在 VS 订阅的页面中点击 Microsoft 365 对应的“激活”即可。

如遇到上图的情况，请添加备用账号（Alternative Account），并且完善备用账号的微软账号信息。这之后再从 VS 订阅页点击“激活”即可。

获取每月 150 美元的 Azure 额度

如果你的 @studentpartner.com 邮箱还没有在 Azure 中设定为中国，则可直接在 VS 订阅的的页面中点击 Azure 150 美元月信用额度（不要点成另一个 Azure 的权益了）对应的“激活”即可。

由于我不是使用的这个方法，因此这部分也不大清楚。只需记得能选其他国家的地方，一定不要选择中国。

如果你的 @studentpartner.com 邮箱已经在 Azure 中设定为中国，会提示这项服务不在中国提供。

此时需要绑定一个微软账户作为备用账户（Alternative Account），该微软账户要求没有注册过 Azure，或区域设定不是中国。

绑定好后，复制“激活”按钮指向的链接，然后在 Azure 登录备用账户。登录完成后，粘贴刚才复制的链接并访问到一下界面。

第一部分的国家不要选中国，电话随便填一个对应国家的电话即可。第二部分会重新让你再输入一个电话并且验证，此时再输入自己的中国电话号码。最后同意协议即可。

如果第三部分要求绑定信用卡，请检查复制的链接是否是 “Azure 开发测试即用即付订阅” 所指向的链接，因为这个订阅要求绑定信用卡。
