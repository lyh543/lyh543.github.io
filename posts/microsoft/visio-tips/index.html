# 连接点

编辑连接线线条、箭头或点 - Visio - Microsoft Support (opens new window)

连接点是形状上的特殊点，可以向它“粘附”连接线和其他形状。 当将连接线或形状粘附至连接点时，即使某个形状发生移动，该连接线和连接点都保持连接状态。

除了使用系统自带的连接点，我们还可以为自己的图形添加连接点。
