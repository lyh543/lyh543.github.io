# 结论

类型 自带排序 定义cmp结构（为<） greater<int> 定义bool cmp函数（为<）
set 升序 升序 降序 \
priority_queue 降序 降序 升序 \
sort 升序 \ \ 升序

表中 \ 表示这种语法无法适用于该排序。
