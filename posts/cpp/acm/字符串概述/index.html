# 库函数（轮子）

# #include<cstring>

  • sscanf(const char \__source, const char *__format, ...) 只能读一次

  • int strlen(char*)

  • int strcmp(const char *str1, const char *str2) 前者小，返回负数，大，返回正数

  • strcpy

  • char *strcat(char *str1, const char *str2) 将 str2 接到 str1 的结尾，用 *str2 替换 str1 末尾的 '\0' 返回 str1

  • char * strchr(char * string, char ch);//寻找字符串

# #include<string>

<string>里的 std::string 与其大量函数。

#include<string>
string c1("this is a string");
string c2(5,'c'); //c2="ccccc"

//string copy
s2 = c1  + "," + c2;
s2 += c2;

注意区分 assignment 和 copy construction 函数参数为 string 时一定要设为 const string &
