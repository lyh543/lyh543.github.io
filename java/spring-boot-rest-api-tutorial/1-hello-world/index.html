热更新

热更新需要安装 spring-boot-devtools，不过创建项目的时候已经安装过了：

如果是idea，需要改以下两个地方：

  1. 勾选 File > Settings > Compiler > Build Project automatically
  2. 勾选 ctrl + shift + alt + / > Registry > compiler.autoMake.allow.when.app.running

不过还是要写完以后 10s 左右才会热更新，有点慢。

Stack Overflow 上给出了另外一种解决办法，把 Build 绑定到 Ctrl + S。每次写完以后 Ctrl + S 一下就会立即 Build，Spring Boot 检测到变化后就会在 1s 之后热部署。
